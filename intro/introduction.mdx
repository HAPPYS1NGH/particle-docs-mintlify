# Particle Network Documentation

Particle Network provides the infrastructure for building **next-generation Web3 applications**.  
Our stack consists of three core products:

1. **Chain Abstraction**
   Powered by **Universal Accounts**, Chain Abstraction enables a single account and unified balance across multiple chains.  
   - Users can combine assets from any supported ecosystem and spend them anywhere — no bridges or manual switching.  
   - Developers make their dApps chain-agnostic by default.

2. **Social Logins**  
   Wallet creation and access with familiar Web2 credentials (Google, Twitter, Discord, etc.).  
   - Lower onboarding friction and let users start using dApps instantly.  

3. **Account Abstraction (ERC-4337 AA)**  
   Smart account infrastructure following the ERC-4337 standard.  
   - Features include gas sponsorship, batched transactions, and programmable account logic.
