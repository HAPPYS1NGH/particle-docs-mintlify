# Quick start

You can scaffold a new Particle Connectkit app with one of the following commands, using your package manager of choice:

```dash
npm init @particle-network/connectkit@latest
# or
pnpm create @particle-network/connectkit@latest
# or
yarn create @particle-network/connectkit
```

This will prompt you for a project name, generate a new directory containing a boilerplate project, and install all required dependencies.

<Tip>The complete web SDK reference is available at [SDKs (Web)](/api-reference/connect/desktop/web).</Tip>
