## Understanding `buildOrder`

- `buildOrder` allows you to build an order using the [Seaport protocol](https://github.com/ProjectOpenSea/seaport) as an alternative to [seaport-js](https://github.com/ProjectOpenSea/seaport-js). At the moment, this only supports single NFT purchases. It takes the following path parameter:

  - `chainId` - integer.

  And the following body parameters:

  - `fromAddress` - string.
  - `tokenAddress` (contract) - string.
  - `tokenId` - string.
  - `price` - string (Wei).
  - `currency` (optional, ERC20 token) - string, `0x0000000000000000000000000000000000000000` by default.

***
